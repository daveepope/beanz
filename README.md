# beanz

Cognitive debt scoring for AI coding sessions

## Install

```bash
brew tap daveepope/beanz https://github.com/daveepope/beanz.git
brew install beanz
```

From a local clone:

```bash
brew tap daveepope/beanz "file://$(pwd)"
brew install daveepope/beanz/beanz
```

Unreleased `main` (builds from source; requires Rust):

```bash
brew install --HEAD daveepope/beanz/beanz
```
